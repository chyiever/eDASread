eDAS数据格式与数据读取软件使用说明（eDASread 当前版本）

第一章 文档说明

本文档面向 eDAS 数据使用人员、算法开发人员和软件操作人员，说明 eDAS 数据格式、eDASread 的界面结构、交互方式、滤波与导出规则。本文以当前代码实现为准。

第二章 eDAS 数据格式说明

2.1 文件命名要求
- 文件扩展名必须为 `.bin`。
- 文件名必须包含采样率字段，例如 `-2000Hz-`。
- 文件名必须包含点数字段，例如 `-0130pt-`。
- 文件名末尾时间戳可选，例如 `-20260410T153045.123.bin`。

2.2 存储结构与类型
- 原始数据类型：`int32`（4 字节）。
- 单文件逻辑矩阵：`frame_count x points`。
- 磁盘存储顺序：按帧连续（frame-major）。

2.3 单位换算
- 显示/滤波前将原始值换算为相位弧度：
  `phase_rad = raw_int32 x (pi / 32767)`。
- 导出滤波后结果时会转换回 `int32` 并写回 bin 文件。

第三章 软件界面与功能

3.1 主界面布局
- 顶部标题栏：左侧显示软件 logo，中部显示标题。
- 状态栏：最右侧固定显示“中国科学院半导体所开发”。
- 左侧：文件浏览区（分页、跳页、home/end）。
- 右侧：参数区 + 波形/时空图区。

3.2 参数区布局（当前版本）
- 两列布局：
  - 左列 `Display Range`。
  - 右列 `Filter + Export`。
- `Display Range` 中依次包含：
  - Point Start / Point End
  - VMin / VMax
  - Colormap
  - Actions 行：Apply / Reset / Reset View / Back View / Zoom Out 2x / 指定位置点时域绘图

3.3 时域图与 time-space 图
- 上方为“指定位置点时域波形”图。
- 下方为 time-space 图（右侧为色标）。
- 两图共用时间轴范围（联动）。
- 两图均支持缩放与框选浏览。

3.4 指定位置点输入框交互
- 仅允许整数。
- 必须位于 `[Point Start, Point End]`。
- 输入框聚焦时可用鼠标滚轮按步长 ±1 调整。

3.5 滤波功能
- 支持 `none / lowpass / highpass / bandpass`。
- 频率约束遵循 Nyquist（采样率一半）。
- 滤波沿时间轴对每个空间点执行。

3.6 导出规则
- 导出对象是当前可视窗口，不是全文件。
- 横向范围按时间换算为帧索引。
- 纵向范围按点索引截取。
- 导出文件命名规则：
  `eDAS-<sample-rate>Hz-<point-count>pt-<start-time>.bin`

第四章 操作流程

1. 启动软件并选择数据目录。
2. 在左侧列表选择文件。
3. 设置点范围、色标范围、colormap、滤波参数。
4. 在“指定位置点时域绘图”输入框中输入或滚轮调整点号。
5. 点击 Apply 更新显示。
6. 使用鼠标缩放、Reset View、Back View、Zoom Out 2x 浏览。
7. 选择导出目录，点击 Export Current View。

第五章 常见问题

- 无法识别采样率/点数：文件名缺少 `Hz` 或 `pt` 字段。
- Invalid point range：Point Start/End 设置越界或顺序错误。
- 指定位置点无效：不是整数或不在 Point Start~Point End 范围。
- 导出为空：当前视窗没有有效帧或点。
